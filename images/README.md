# طريقة تغيير صور المنتجات

كل منتج بالموقع يبحث تلقائياً عن صورة بهذا المسار:
`images/اسم-الصنف.jpg`

لتغيير أو إضافة صورة منتج:
1. صوّر المنتج بشكل مربع (نفس الطول والعرض تقريباً).
2. سمّي الصورة بنفس الاسم المطلوب بالضبط (بالأسفل)، بامتداد jpg.
3. ارفعها داخل مجلد images بنفس اسم الريبو.
4. خلاص — بتظهر بمكان الأيقونة تلقائياً بدون أي تعديل بالكود.

إذا الصورة غير موجودة، يبقى يظهر مكانها تصميم بديل بلون العنابي وأيقونة — الموقع ما ينكسر أبداً.
